# tree_visualizer_cpp
Visualize a tree in cpp :/

## How to use

Compile and run `tree_visualizer.cpp`, you can manually type the tree or import throught `tree.txt` file

### tree.txt format
`n` - <number of nodes>
**Followed by `n` lines**
`v` - <value of node> `p` - <parent of node (by input position, start from 0)>
